Pango::Cairo
=======

Perl bindings to the PangoCairo library

INSTALLATION
------------

To install this module type the following:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install


DEPENDENCIES
------------

This module requires these C libraries:

  * glib

  * cairo

  * pangocairo

and these Perl modules:

  * Glib::Object::Introspection

  * Cairo
